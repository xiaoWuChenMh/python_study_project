### 判断任务是否激活


**任务梳理**

```text
-- 变量 --
重试次数=3（默认为3）

-- 逻辑 --
判断任务列表第一项是否领取了任务：（代表已经激活）点击任务栏
从活动页领取任务，一次完成是否也可以这样重新领取
需要重试3次
   
```


### 激活后的流程

```text

-- 变量 --
副本标识=0
is_sleep = True

-- 逻辑 --
是否使用道具：
   是否刺杀，点击，并按技能键
   其他道具，睡（道具使用耗时）
是否有npc对话框：点击
底层npc文案：点击（这种是否有倒箭头标识）
是否最后的任务：
    打：直接打，并按技能键
    跑：需要规范跑的路线

```
